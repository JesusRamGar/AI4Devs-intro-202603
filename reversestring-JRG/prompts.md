# Prompts

## Initial Prompt
Create a complete solution using index.html and script.js to reverse a user-entered text string. The index.html must match the UI of the image.

## Final Prompt
Create a complete solution using index.html and script.js to reverse a user-entered text string.

Requirements:
- The UI in index.html must match the provided image.
- All JavaScript logic must be implemented in script.js.
- Do not include inline JavaScript in the HTML.
- The page must include:
    - A title "Reverse String"
    - An input field
    - A button to reverse the text
    - A section to display the result
    - A button to copy the result to clipboard
- The reverse functionality must correctly handle any text input.

Additional constraints:
- The input must not be empty.
- The input length should be limited to a reasonable size (e.g., 1000 characters) to avoid performance issues.
